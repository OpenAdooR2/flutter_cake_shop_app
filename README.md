# 🍰 Cake Finder — Flutter App

แอปพลิเคชันสำหรับค้นหาและแนะนำร้านเค้กยอดนิยม  
พัฒนาด้วย **Flutter** เน้นความสวยงาม ใช้งานง่าย และตอบโจทย์สายคาเฟ่

---

## 🚀 Overview

Cake Finder ช่วยให้คุณค้นหาร้านเค้กได้ง่ายขึ้น พร้อมข้อมูลครบ:
- รูปภาพร้าน
- เวลาเปิด-ปิด
- รายละเอียด
- แผนที่
