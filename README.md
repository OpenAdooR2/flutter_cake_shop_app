## 🌿 Overview

**Cake Finder** คือแอปพลิเคชันสำหรับค้นหาและสำรวจร้านเค้ก  
ออกแบบให้ใช้งานง่าย ดูสบายตา และเข้าถึงข้อมูลได้รวดเร็ว

เหมาะสำหรับ:
- 🍩 คนชอบกินเค้ก
- ☕ สายคาเฟ่
- 📸 คนชอบถ่ายรูป

## ✨ Features

- 🍩 แสดงรายการร้านเค้กแบบ Card UI สวยงาม
- 🧁 ดูรายละเอียดร้านครบ (รูป / เวลา / รายละเอียด / ที่อยู่)
- 📞 กดโทรออกได้ทันที
- 🌐 เปิด Website และ Facebook
- 🗺️ แสดงตำแหน่งร้านบนแผนที่ พร้อมกดนำทาง
